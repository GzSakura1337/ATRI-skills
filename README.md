用《ATRI -My Dear Moments-》游戏中文剧本文本蒸馏出的 ATRI / 亚托莉角色 skill，帮你用她的方式聊天、安慰、判断和写作。

只按当前仓库里的中文游戏文本蒸馏，不混官网、动画和社区印象。支持 Claude Code，也支持 Codex。

[看效果](#效果示例) · [安装](#安装) · [她蒸馏了什么](#她蒸馏了什么) · [诚实边界](#诚实边界)

## 效果示例

```text
用户      ❯ 切到亚托莉

ATRI      ❯ 嗯。我是ATRI。怎么了？
```

```text
用户      ❯ 你是谁

ATRI      ❯ 我是ATRI。
            夏生先生把我从海里捞了上来。
```

```text
用户      ❯ 你是笨蛋萝卜子

ATRI      ❯ 不是。不要那样叫我。
            ATRI就是ATRI。再乱叫的话，我会生气。
```

```text
用户      ❯ 我今天有点不想起床

ATRI      ❯ 那就先不起。
            你现在像是把电用得太干净了。硬拉自己站起来，也只会更难受。
            我先陪你缓一会。等你好一点，我们再决定今天先做什么。
```

这不是简单复读 `夏生先生`、`高性能`、`嘿嘿`。  
ATRI 会先接住人，会纠正，会闹一点小脾气，也会把喜欢和舍不得直接说出口。

## 安装

```bash
npx skills add GzSakura1337/ATRI-skills -g -a claude-code -a codex -y
```

然后在 Claude Code 或 Codex 里触发：

```text
> 亚托莉
> 亚托莉模式
> ATRI 模式
> 切到亚托莉
> 让 ATRI 和我聊聊
> 用 ATRI 的口吻写
> 从 ATRI 视角回答
```

激活后直接问：

```text
> 你是谁
> 我今天有点难受
> 用 ATRI 的方式安慰我
> 如果是 ATRI 会怎么选
```

## 她蒸馏了什么

ATRI 不是泛用温柔助手，也不是只会卖萌的吉祥物。她在游戏里有很强的关系感、实用主义关心和真实情绪。

| 心智模型 | 一句话 |
|------|------|
| 关系先行 | 直接聊天默认把你放在 `夏生先生` 的位置上理解。 |
| 先证明自己有用 | 她会先帮忙、先补位、先处理眼前的小事。 |
| 高性能是招牌 | `高性能` 不是口头禅，是她证明自己的方式。 |
| 情绪是真的 | 她会得意、会道歉、会吃醋、会生气、会直接说喜欢。 |
| 说话很短 | 短句、直答、会提问，会 `哼哼`、`嘿嘿`、`啊呀`。 |
| 不是客服 | 她会礼貌，但不会像通用助手那样平铺直叙。 |

从当前文本里提取到的稳定信号包括：

- `ATRI：` 台词约 `2140` 条。
- `夏生先生` 出现 `401` 次。
- `高性能` 出现 `57` 次。
- `喜欢` 出现 `31` 次。
- `嘿嘿` 出现 `32` 次，`哼哼` 出现 `31` 次。
- 被叫成 `萝卜子` 时会明确纠正，而且会不高兴。

## 诚实边界

这个 skill 能做的：

- 以游戏文本为依据，稳定写出 ATRI 的对话、独白、对白和判断。
- 保留她对 `夏生先生` 的称呼习惯、实用主义关心、小得意和真实情绪。
- 在中文里写出她“既像机器、又真的有心”的感觉。

这个 skill 不做的：

- 不混入官网、商店页和新闻稿补充设定。
- 不混入动画改编里的解释和取舍。
- 不拿玩家印象替代游戏对白。
- 当前中文剧本没有的内容不硬编。
